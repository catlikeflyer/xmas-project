# Under the Missile Toad
Simple game made with HTML and vanilla JavaScript to conmemorate Christmas. Mobile-first design.

## How to play
- Click on the falling missile toads before they reach the end of the screen to score points. 
- If a toad reaches the end of the screen, you lose.

## Functionality
- Leaderboard
- Sound effects
- Mobile-first design
- Firebase (Firestore) integration
